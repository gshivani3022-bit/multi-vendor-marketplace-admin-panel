🛒 Multi-Vendor Marketplace Admin Panel
🎯 Objective
The main goal of this project is to help students understand:

How an e-commerce admin dashboard is designed
How to build responsive layouts
How to implement basic front-end interactivity
✨ Features
📊 Dashboard Overview

Displays key metrics like revenue, orders, and users
Uses cards and charts for visualization
🏪 Vendor Management

View vendor details
Approve or reject vendors
📦 Product Management

Add, edit, or delete products
Display products in structured tables
🧾 Order Management

Track order details
View order status
👥 Customer Management

Display customer data
Manage user records
✅ Approvals Section

Handle vendor/product approvals
📱 Responsive Design

Works on desktops, tablets, and mobile devices
